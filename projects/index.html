Major Projects at the University of Minnesota since 1999

  • Asset Tracking

    1st Gen: 1999; MS Access, SQL Server
    2nd Gen: 2007; JSF, SQL Server
    3rd Gen: 2011; Grails, SQL Server
    In-house application used to track staff and equipment information such as locations, usage dates and contract details.

  • Bin Mover

    2001; ASP, T-SQL, SQL Server Enterprise application used by the division to track upwards of 100,000 patient charts across campus and hospitals. The application also serves as a work flow system to track study work flow processes and ensure that cases are routed according to study protocol.

  • Diagnostic Treatment and Followup System

    2001; ASP, T-SQL, SQL Server Enterpise application used by multiple large-scale cancer prevention studies to track cancer screening results, initiate followup, manage medical records requests, reminders, and screening results delivery to study subjects and primary care physicians.

  • Study Participant Lookup

    2000; ASP, T-SQL, SQL Server Search and reporting system to facilitate locating study subjects from a database over over 1.5 million people, and providing person-centric drill down views of all study data gathered during the years of enrolment for the particular study participant.

  • Duplicate Checker

    C#, ASP, T-SQL, SQL Server 2003; Highly optimized system to match incoming participant recruitment data to match people against existing person records with the database of over 1.5 million people to prevent duplication of people, and to prevent multiple enrollment within a study.

  • Medical Records Authorization Tracking

    2003; C#, T-SQL, SQL Server Used to generate, track and receipt medical records authorization required by study protocol, HIPAA and other regulatory requirements for multiple studies, some of them tracking over 20,000 participants.

  • Document Generation ncs-grails/ncs-case-management

    1st Gen: 2004; MS Access, SQL Server
    2nd Gen: 2007; C#, SQL Server
    3rd Gen: 2011; Grails, MySQL Server
    System to automate the production and tracking of hundreds of types of document bundles such as surveys, calling forms and recuitment letters often times containing upwards of hundreds of thousands of documents per batch. The system has been used to generate over 500 different instrument types for over 100,000 different recipients and has generated over 1 million unique documents, all of which are tracked and regeneratable within the system.

  • Appointment Scheduling

    2004; Visual Basic 6.0, SQL Server System used to schedule hundreds of thousands of cancer screening appointments for many different studies.

  • Incentive Reimbursement

    2004; ASP, SQL Server Used to track the acceptance and distribution of various kinds of study incentives such as taxi vouchers, milage reimbursement, and cash incentives across multiple studies.

  • Secure Messaging ncs-grails/ncs-secure-mail

    1st Gen: 2006; ASP, SQL Server
    2nd Gen: 2011; Grails, MySQL Server
    Web based system to provide a mechanism that ensures the secure transmission of sensitive data as an alternative to insecure email.

  • Participant Self Enrollment ncs-grails/ncs-signup

    2011; Grails, MySQL Server Used to allow potential study participants to request enrollment via a secure web based system.

  • Online Call Tracking ncs-grails/ncs-phone

    1st Gen: 2005; ASP, T-SQL, SQL Server
    2nd Gen: 2011; Grails, MySQL Server
    System to manage instruments within a call center including tracking incoming and outgoing phone calls, call results, adminstrative tasks, instrument completion formation and calling queues.

  • Demographic Management

    1st Gen: 2003; MS Access, SQL Server
    2nd Gen: 2006; Visual Basic 6.0, T-SQL, SQL Server
    3rd Gen: 2012; Grails, SQL Server
    System to manage the entry of demographic details and contact information for thousands of active study participants and potential recruits.

  • Slide Pathology Scanning and Tracking

    1st Gen: 2008; JSF, SQL Server
    2nd Gen: 2010; Grails, SQL Server
    Application used to track the receipting of brains along with the cutting, slicing and staining of tissue samples of various brain regions to tie the original brain region source to the resultant ultra-high resolution scans of the slides via integration with an Aperio digital pathology system.

  • E.T.L Processes and Automation ncs-grails/ncs-etl

    Visual Basic 6.0, C#, BASH, T-SQL, Grails, SQL Server MySQL Server The creation of dozens of ETL processes throughout the years to automate the migration of data from sources ranging from incoming emails to XML data posted to RESTful web services into production database systems.

  • X-Ray Scanning and Management

    2007; C#, SQL Server Managed the scanning and QA process for over 10,000 X-Rays into a digital library system to aid in the creation of an automated X-Ray diagnosis and tumor detection system.

  • Clinic Chart Scanning

    2011; BASH, Python, Linux based high-speed image scanning and processing system used to scan, post-process, and OCR over a half a million pages of clinic charts.

  • Obituary Parsing and Notification

    2004; BASH, T-SQL, SQL Server Automated process to load obituary data from many national and local newspapers to be parsed and standardized to automatically identify deceased study participants nightly for large studies.
